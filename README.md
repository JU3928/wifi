# WiFi 密码破解工具

纯静态网页工具 —— 输入 WiFi 名称，下载自包含破解脚本，双击运行，自动遍历密码尝试连接。

🌐 **线上地址**：https://ju3928.github.io/wifi/

## 如何使用

1. 浏览器打开 https://ju3928.github.io/wifi/
2. 填写目标 WiFi 名称（SSID）
3. 点击下载，得到一个 `.bat` 文件
4. 在 Windows 电脑上**双击运行**
5. 脚本自动破解：
   - **阶段1**：常见密码（~150条）
   - **阶段2**：全部8位数字组合（00000000~99999999）
6. 找到密码输入则显示，全部失败则提示"已尽力了"

## 依赖

**零依赖**。无需 Python、无需 Node.js、无需任何安装。
脚本调用 Windows 自带的 `netsh wlan` 命令。
